# AI 标注组件使用指南

## 📋 概述

新增了三个 AI 标注相关的组件，用于构建图像标注应用。这些组件提供了完整的图像上传、AI 预标注和手动标注功能。

---

## 🎯 组件列表

### 1. 图像上传（ImageUpload）

**功能描述：**
- 支持拖拽上传图片
- 支持点击选择图片
- 实时预览上传的图片
- 文件大小和格式验证

**默认属性：**
```typescript
{
  maxSize: 5,              // 最大文件大小（MB）
  accept: 'image/*',       // 接受的文件类型
  multiple: false,         // 是否支持多选
  showPreview: true        // 是否显示预览
}
```

**使用场景：**
- 图像标注项目的第一步
- 需要批量上传图片进行处理
- 图片预处理和质量检查

---

### 2. AI 预标注（PreAnnotation）

**功能描述：**
- 模拟 AI 自动检测和标注
- 显示处理进度
- 展示检测结果列表
- 支持重新标注

**默认属性：**
```typescript
{
  title: 'AI 预标注',     // 组件标题
  autoStart: false        // 是否自动开始标注
}
```

**检测结果包含：**
- 目标类别标签
- 置信度分数
- 边界框坐标

**使用场景：**
- 大规模图像数据的初步标注
- 辅助人工标注提高效率
- 质量检测和分类

---

### 3. 标注画布（AnnotationCanvas）

**功能描述：**
- 交互式图像标注界面
- 支持矩形、圆形、多边形标注
- 实时绘制和预览
- 标注编辑和管理

**默认属性：**
```typescript
{
  title: '标注画布',      // 组件标题
  imageUrl: '...',        // 待标注图像URL
  width: 800,             // 画布宽度
  height: 600             // 画布高度
}
```

**功能特性：**
- ✅ 多种标注工具（矩形/圆形/多边形）
- ✅ 自定义标注颜色
- ✅ 缩放功能（50%-200%）
- ✅ 撤销/清空操作
- ✅ 保存标注数据

**快捷操作：**
- 鼠标拖动：创建标注框
- 撤销按钮：删除最后一个标注
- 清空按钮：删除所有标注
- 保存按钮：导出标注数据

## 🚀 使用流程

### 完整的标注工作流：

```
1. 拖拽 ImageUpload 组件
   ↓
   上传待标注图像
   ↓
2. 拖拽 PreAnnotation 组件
   ↓
   运行 AI 预标注
   ↓
   查看检测结果
   ↓
3. 拖拽 AnnotationCanvas 组件
   ↓
   手动调整/添加标注
   ↓
   保存最终标注数据
```

## 📝 注意事项

1. **图像上传**：
   - 建议图片大小不超过 5MB
   - 支持常见图像格式（jpg, png, gif, webp）

2. **AI 预标注**：
   - 当前为模拟演示功能
   - 实际项目中需要接入真实 AI 模型 API

3. **标注画布**：
   - 标注数据保存在浏览器内存中
   - 需要点击"保存"按钮导出数据
   - 建议定期保存标注结果
